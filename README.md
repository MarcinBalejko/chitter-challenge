Chitter Challenge
=================

Building a small Twitter clone that will allow the users to post messages to a public stream.

Tools and technologies used:

* Ruby / Sinatra
* PostgreSQL
* RSpec / Capybara
* HTML / CSS
* Bootstrap

**Aiming for:**

* Complete TDD process ✔
* 100% test coverage ✔
* User interface which resembles original Twitter ✔
